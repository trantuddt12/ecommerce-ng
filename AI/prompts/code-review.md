# Prompt Mau: Code Review

## Muc dich

Dung khi can AI review thay doi code trong repo nay theo huong tim loi va regression.

## Prompt

```text
Hay review thay doi code trong Angular project nay voi uu tien sau:
1. Bug va regression hanh vi.
2. Route, guard, auth flow, interceptor, state issues.
3. Sai convention hoac mo rong sai vi tri trong core/features/shared.
4. Missing validation, loading, error handling.
5. Testing gap quan trong.

Yeu cau tra ve:
- Liet ke finding truoc, nghiem trong nhat len tren.
- Moi finding nen co file va dong neu co the.
- Neu khong co finding ro rang, noi ro khong co finding va liet ke residual risks.
```
