# Prompt Mau: Cap Nhat Tai Lieu

## Muc dich

Dung khi can AI cap nhat tai lieu sau mot thay doi kien truc hoac feature.

## Prompt

```text
Hay cap nhat tai lieu cho Angular project nay dua tren code hien tai.

Pham vi cap nhat:
<liet ke feature, route, auth flow, convention, hoac files vua doi>

Yeu cau:
1. Bam source code that trong repo, khong mo ta chung chung.
2. Cap nhat reference truoc neu thay doi kien truc hoac luong chay.
3. Cap nhat instruction neu thay doi convention.
4. Cap nhat prompt/skill neu xuat hien tac vu lap lai moi.
5. Viet ngan gon, de tra cuu, co lien he ro voi file code.
```
