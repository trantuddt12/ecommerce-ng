# Prompt Mau: Dieu Tra Bug

## Muc dich

Dung khi can AI tim root cause va sua loi trong codebase nay.

## Prompt

```text
Ban dang debug mot Angular project co auth flow, guards, interceptors, signal store, va SSR.

Van de:
<mo ta bug hoac log loi>

Yeu cau:
1. Tim file va luong chay lien quan truoc khi de xuat cach sua.
2. Uu tien tim root cause that su, khong sua bua vay.
3. Kiem tra route, API call, store, interceptor, va component lifecycle neu lien quan.
4. Chi sua pham vi toi thieu de giai quyet bug.
5. Neu khong the xac nhan day du, ghi ro gia dinh va testing gap.

Dau ra mong muon:
- Nguyen nhan goc.
- File da sua.
- Cach verify.
```
