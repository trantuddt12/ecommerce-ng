# Documentation Rules

## Nguyen tac viet tai lieu

- Viet ngan gon, co cau truc, bam code that.
- Uu tien mo ta cach he thong dang chay, khong viet mong muon ly tuong.
- Moi tai lieu nen tra loi duoc: dung de lam gi, doc khi nao, lien quan file nao.

## Muc toi thieu cho tai lieu ky thuat

- Muc dich.
- Pham vi.
- File hoac thanh phan lien quan.
- Luong xu ly chinh.
- Rui ro hoac ghi chu quan trong.

## Muc toi thieu cho prompt/skill

- Khi nao dung.
- Dau vao can cung cap.
- Dau ra mong muon.
- Rang buoc can ton trong.

## Nguon su that uu tien

1. Source code trong `src/app/`.
2. Cau hinh project trong `package.json`, `angular.json`.
3. Tai lieu noi bo trong `doc/`.
4. Tai lieu AI trong `AI/`.
