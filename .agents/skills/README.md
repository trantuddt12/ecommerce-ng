# Installed CLI Skills

Thu muc nay duoc quan ly boi `skills` CLI.

- Day la noi chua cac skill cai dat ben ngoai de agent nap truc tiep.
- Khong nen dat tai lieu noi bo cua du an vao day.
- Tai lieu workflow noi bo cua repo nam o `AI/workflows/`.

Neu can them skill moi, dung `npx skills add ...` de cap nhat dong bo cung `skills-lock.json`.
